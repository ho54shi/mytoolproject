NERのためのアノテーションツール

### 概要
- 固有表現にラベル付けをする目的を想定したツールです。
- 事前学習したモデルで固有表現の候補を提示し，ユーザーはそれら修正し，アノテーションの効率を向上させます。
- ユーザーが提出したアノテーションデータを使用し画面上からモデルを学習させることができます。


### 使い方
- ユーザーは新規登録します。登録時に自身が専門家か否かを登録します。登録済みの方はログインします。
- アノテーションを施したいプレーンテキストをアップロードします。拡張子は「.txt」のみに制限されています。
- テキストをアップロードした後は，つけたいラベルを作成します。ラベルごとに，ラベル名，色，ショートカットキーの３つを設定できます。
- 現在は，ルート権限があるユーザーが設定したラベルを全ユーザーが取得できるため，各ユーザーがそれぞれ設定する必要はありません。
- 各プロジェクトの「詳細」をクリックすると，詳細情報とともに改行ごとにラベル付けできるリストが表示されます。
- 「ラベル付けする」ボタンをクリックして，ラベル付け作業に移ります。
- ラベル付けしたいテキストを選択した後に，ラベル付けしたいラベルのボタンをクリックするか設定したショートカットキーを入力してラベル付けを行います。ラベルを消したい場合は各ラベルの消去ボタンをクリックして消します。
- 作業が終了したら下の「submit」ボタンを押します。確認画面が表示されるので，問題がなければ「OK」をクリックするかEnterキーを押します。
- プロジェクト詳細画面に戻るので，別の文で作業するか別プロジェクトに移ります。
- 固有表現抽出モデルを学習させたい場合，「学習ページへ」をクリックし，「学習開始」ボタンをクリックします。
このとき，すでに学習スクリプトが動いている場合，その旨が表示されするので待つかアノテーション作業に戻ります。
- 学習スクリプトが動いていない場合，「学習開始」ボタンを押し，さらに「プロジェクトに戻る」ボタンを押してアノテーション作業に戻ります。
- すべての作業が終了したらログアウトします。
